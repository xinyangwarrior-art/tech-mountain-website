# Tech Mountain Website Starter Template

This folder contains a polished website starter for Tech Mountain that prioritizes small-to-medium business IT support and lead generation.

## What this template does

- Provides a clean modern homepage layout built with React and Vite.
- Uses a calm, trustworthy blue-and-white visual system with strong business-focused messaging.
- Keeps content easy to update by centralizing copy in `src/data/content.js`.
- Includes a design documentation folder for brand voice, site structure, and future expansion.

## Where to edit

- `src/App.jsx` — routing and page structure.
- `src/pages/` — detail page content for Managed IT and Security.
- `src/data/content.js` — homepage content and service text.
- `src/styles/main.css` — colors, spacing, layout, and typography.
- `src/components/` — component-based section templates.

## Run locally

From `site/`:

```bash
npm install
npm run dev
```

## Future expansion

- Add service detail pages under `src/pages/`.
- Add a CMS or markdown-driven content layer later.
- Add a contact form backend or lead capture workflow.
